Адвокатская практика

Ниже приведён краткий обзор некоторых дел, в которых я участвовал по поручению своих доверителей в качестве представителя истцов, ответчиков, третьих лиц по делам, связанным с гражданскими, семейными, жилищными, трудовыми, наследственными, земельными и иными правоотношениями, защитника лиц, привлекаемых к административной ответственности, представителя потерпевших по административным делам, защитника подозреваемых, обвиняемых, подсудимых, осужденных, представителя потерпевших по уголовным делам, представителя взыскателей по делам, связанным с исполнительным производством.

Гражданские дела, связанные с жилищным законодательством

  • гражданское дело по иску о признании права собственности в порядке приватизации на служебное жилое помещение, процессуальный статус — представитель истцов, третьего лица;
  • гражданское дело по иску о признании права собственности в порядке приватизации на жилое помещение, предоставленное КЭЧ Министерства Обороны РФ, процессуальный статус — представитель истца;
  • гражданское дело по иску о сохранении жилого помещения в перепланированном и переустроенном состоянии, процессуальный статус — представитель истца;
  • гражданское дело по иску о признании права собственности на измененный в процессе реконструкции объект недвижимости, процессуальный статус — представитель истца;
  • гражданские дела по искам о признаниигражданина утратившим права пользования жилым помещением и снятии с регистрационного учета, процессуальный статус — представитель истцов, ответчиков;
  • гражданское дело по иску о признании гражданина членом семьи умершего нанимателя жилого помещения, признании гражданина последующим нанимателем жилого помещения и обязании органа местного самоуправления заключить договор социального найма в письменной форме, процессуальный статус — представитель истца;
  • гражданские дела об определении порядка пользования жилыми помещениями, процессуальный статус — представитель истцов, ответчиков;
  • гражданские дела о взыскании компенсации за пользование имуществом, находящимся в долевой собственности, процессуальный статус — представитель истцов, ответчиков;
  • гражданские дела о вселении, нечинении препятствий в пользовании жилым помещением, процессуальный статус — представитель истцов, ответчиков;

Гражданские дела, связанные с земельным законодательством

  • гражданское дело по иску о признании незаконной постановку земельного участка на государственный кадастровый учет, установлении границ, процессуальный статус — представитель ответчика;
  • гражданское дело по иску об оспаривании границ земельного участка, процессуальный статус — представитель истца;
  • гражданское дело по иску о понуждении заключить договор аренды земельного участка с множественностью лиц на стороне арендатора, процессуальный статус — представитель истца;
  • гражданские дела по искам об определении порядка пользования земельными участками, процессуальный статус — представитель истцов, ответчиков;

Гражданские дела, связанные с семейным законодательством

  • гражданские дела по искам о взыскании алиментов на содержание супруга, процессуальный статус — представитель истцов, ответчиков;
  • гражданские дела по искам о расторжении брака и разделе совместно нажитого имущества, процессуальный статус — представитель истцов, ответчиков;
  • гражданское дело по иску о порядке общения и осуществления права на воспитание ребенка при раздельном с ним проживании, процессуальный статус — представитель истца;
  • гражданские дела по искам о признании долга одного из супругов общим долгом супругов, процессуальный статус — представитель истцов, ответчиков;
  • гражданские дела по искам о взыскании алиментов на содержание ребенка (детей), процессуальный статус — представитель истцов, ответчиков;
  • гражданские дела по искам о взыскании неустойки в связи с образованием задолженности по уплате алиментов на содержание ребенка (детей), процессуальный статус — представитель истцов, ответчиков;

Гражданские дела, связанные с наследством

  • гражданское дело по иску о признании права собственности на долю квартиры в порядке наследования, процессуальный статус — представитель истца;
  • гражданское дело по иску о восстановлении срока для принятия наследства, признании наследника принявшим наследство в порядке наследственной трансмиссии, определении долей наследников в наследственном имуществе, признании недействительным свидетельства о праве на наследство, процессуальный статус — представитель истца;
  • гражданское дело особого производства по заявлению об установлении места открытия наследства, процессуальный статус — представитель заявителя;
  • гражданское дело особого производства по заявлению об установлении факта родственных отношений, процессуальный статус — представитель заявителя;
  • гражданское дело об установлении факта непринятия наследства наследником первой очереди, об установлении факта принятия наследства наследником второй очереди по праву представления, о признании права собственности на долю в квартире в порядке наследования по закону, процессуальный статус — представитель истца;

Гражданские дела, связанные со страховым законодательством

  • гражданские дела по искам о взыскании со страховщиков не произведенной страховой выплаты (ее части)по договору обязательного страхования (ОСАГО), неустойки за нарушение страхового законодательства, штрафа за нарушение прав потребителя, процессуальный статус — представитель истцов;
  • гражданские дела по искам о взыскании со страховщиков страхового возмещения, процентов за пользование чужими денежными средствами, законной неустойки за нарушение срока оказания услуги потребителю по договору добровольного страхования по рискам Автокаско (Угон+Ущерб), штрафа за нарушение прав потребителя, процессуальный статус — представитель истцов;
  • гражданские дела по искам о взыскании со страховщиков страхового возмещения, процентов за пользование чужими денежными средствами вследствие просрочки в их уплате по договору добровольного страхования по рискам Автокаско (Угон+Ущерб), процессуальный статус — представитель истцов;
  • гражданские дела, возникающие из договоров страхования жизни, здоровья и трудоспособности, процессуальный статус — представитель истцов;

Гражданские дела, связанные с трудовым законодательством

  • гражданские дела по искам об установлении факта трудовых отношений, внесении записей в трудовую книжку, взыскании задолженности по выплате заработной платы и других невыплаченных денежных средств, компенсации морального вреда, процессуальный статус — представитель истцов;
  • гражданские дела об оспаривании примененных работодателем дисциплинарных взысканий, процессуальный статус — представитель истцов;
  • гражданские дела о признании увольнения незаконным, восстановлении на государственной гражданской службе, взыскании утраченного денежного содержания за время вынужденного прогула, компенсации морального вреда;
  • гражданские дела о взыскании с работодателя невыплаченной заработной платы, в том числе в части невыплаченной заработной платы за сверхурочную работу;

  • гражданские дела по искам таможенных органов о взыскании таможенных платежей за передачу временно ввезенных транспортных средств другим лицам без разрешения таможенного органа, процессуальный статус — представитель ответчиков;
  • гражданские дела по заявлениям об оспаривании решения таможенного органа по результатам камеральной таможенной проверки, признании незаконным отказа таможенного органа в предоставлении возможности ознакомиться с материалами камеральной таможенной проверки, процессуальный статус — представитель заявителей;

Гражданские дела, связанные с корпоративным правом

  • гражданские дела о взыскании денежных средств в счет уменьшения цены некачественно выполненных работ по договору строительного подряда, убытков, понесенных в связи с нарушением прав потребителя, неустойки, компенсации морального вреда, процессуальный статус — представитель истцов;
